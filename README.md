## KYC Backed API

Installation process

- PHP 8.2.12
- Composer version 2.7.7
- PostgreSQL 16.3
- Laravel v11.15.0

Above stack using for application.

## php.ini file settings

should be enabled in php.ini
- extension=pdo_pgsql
- extension=pgsql
  
cd kyc_backend
- composer install
- php artisan jwt:secret
